# Web Drug Twinn - TB Therapy Monitoring

AI-powered web application for TB (Tuberculosis) therapy monitoring.

## Features
- Multi-role system (Patient, Nurse, Admin, Doctor)
- Complete TB clinical data management
- AI-powered alerts and risk assessment
- Medication reminders and alarms
- Real-time monitoring dashboard

## Tech Stack
- Next.js 15 (App Router)
- Tailwind CSS + Shadcn/UI
- Neon Postgres (Serverless)
- Drizzle ORM
- Vercel deployment

## Quick Start (Local)
```bash
cd web-drug-twinn
npm install
npm run dev
```

## Deployment
1. Push to GitHub
2. Connect to Vercel
3. Add Neon DATABASE_URL env var
4. Run `npm run db:push`
